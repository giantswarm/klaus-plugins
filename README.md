# Giant Swarm Klaus Plugins

Klaus plugins for Giant Swarm customers.

## Plugins

### base

A base Klaus plugin with common commands.

**Commands:**

- `/base:hello` - Greet the user with something unexpected

---

## Installation

Add this repository as a marketplace to Klaus. In Klaus, execute:

```nohighlight
/plugin marketplace add giantswarm/klaus-plugins
```

Then use the `/plugin` command to install, enable, disable plugins.
